## 🚀 Quick Start (3 Steps)

### Step 1: Navigate to Bookkeeping
1. Log in to Atonix Capital
2. Go to **Enterprise → Entities**
3. Click any entity (e.g., "Atonix Nigeria Ltd")
4. Click **"📚 Bookkeeping"** tab
5. Click **"Open Bookkeeping Dashboard"**

### Step 2: Set Up (Auto-Generated)
When you create a new entity, the system automatically creates:
- ✅ 6 income categories
- ✅ 12 expense categories
- ✅ 1 default bank account
- ✅ All departments & roles

**Nothing to set up manually!**

### Step 3: Record Your First Transaction
1. Click **"+ New Transaction"** button
2. Fill in the form:
   - **Type**: Income or Expense
   - **Category**: Auto-populated based on type
   - **Account**: Select account
   - **Amount**: Enter amount
   - **Description**: What was this for?
   - **Date**: Transaction date
3. Click **"Save Transaction"**
4. ✨ **Done!** Balance updates automatically

## 📈 Understanding Your Numbers

### Net Profit Formula
```
Net Profit = Total Income - Total Expenses
```

**Example:**
- Income: ₦500,000
- Expenses: ₦120,000
- **Net Profit: ₦380,000** ✅ (Positive = Good!)

### When Net Profit is Negative
```
Net Profit = ₦100,000 - ₦150,000 = -₦50,000 ⚠️
```
This means you spent more than you earned. Time to:
- Reduce expenses
- Increase income
- Review budgets

### Payroll Total
- Sums all transactions where category contains "Salary"
- Helps track personnel costs
- Useful for HR and finance planning

## ❓ FAQ

**Q: Can I edit a transaction after saving?**  
A: Yes! Click on any transaction to edit. Changes are logged in audit trail.

**Q: Can I delete a transaction?**  
A: Yes, but it's logged. Account balance reverses automatically.

**Q: Are account balances calculated automatically?**  
A: Yes! Balance = Previous Balance + Income - Expenses

**Q: Can other entities see my bookkeeping data?**  
A: No! Each entity's data is completely isolated.

**Q: What's the difference between categories and accounts?**  
A: 
- **Categories**: What the money was for (Rent, Salaries, Sales)
- **Accounts**: Where the money came from/went to (Bank, Wallet, Cash)

**Q: Can I export data to Excel?**  
A: Export functionality coming soon. Currently visible in dashboard.

**Q: What happens if I overspend?**  
A: Account balance goes negative. Dashboard shows warning. No blocking - you can still transact.

**Q: How do I track petty cash?**  
A: Create a "Petty Cash" account with type="cash". Record all cash transactions there.
